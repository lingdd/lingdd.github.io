2017计划

2016年终于从学生进入到职场生涯，诚然，学生生涯已经告一段落，而职业生涯才拉开序幕。为了能够让未来的我少一点埋怨，现在需要给自己一个年度奋斗目标。

  1. 保持健康。保持精力充沛能够提高效率，新的一年希望能够通过跑步来保持健康的体魄。
  2. 多读书，不少于30本。书籍乃人生食量，为了提升能力及素养，需要读更多的书，了解更多的事。
  3. 多写作，不少于50篇。写作能够提升自我的表达能力，也可以了解自己过去的思想。
  4. 学一种乐器。音乐🎵能够修身养性。
  5. 结婚计划。ss
